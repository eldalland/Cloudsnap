AWS Services Used

Amazon Cognito

Authenticates users before uploads.

Amazon API Gateway

Receives image upload requests.

AWS Lambda

Processes and resizes images automatically.

Amazon S3

Stores original and optimized images.

AWS Step Functions

Orchestrates multi-platform image resizing.

Amazon CloudFront

Delivers optimized images globally.

Amazon SNS

Sends email completion notifications.

Amazon CloudWatch

Monitors logs, errors, and performance.
